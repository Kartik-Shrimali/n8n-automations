# Contact Form Automation 

## What it does
- Receives contact form submissions via Webhook
- Sends email notification to business owner via Gmail
- Logs the lead in Google Sheets automatically

## Tech Stack
- n8n (automation)
- Gmail API
- Google Sheets API

## How to use
1. Import workflow.json into n8n
2. Set up Gmail and Google Sheets credentials
3. Point your contact form to the webhook URL
